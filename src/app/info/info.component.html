What is Jumbot?

Jumbot is a Discord bot with a Jumbo theme that can help you with your daily tasks. It is designed to be easy to use and to be as user-friendly as possible. The bot is still in development and new features are added regularly.

How do I use Jumbot?

You can use Jumbot like any other Discord bot by using the slash commands. You can find all available commands by starting with / in the chat. (Note: Not all commands are available in DM, some commands require a server to be used in.)
